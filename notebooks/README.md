# Jupyter Notebooks

Welcome to the notebooks section! Here you'll find hands-on learning materials and interactive tutorials.

## Getting Started

### Prerequisites
- Python 3.9+
- Jupyter Notebook or JupyterLab

### Installation

```bash
pip install jupyter notebook
```

Or for JupyterLab:
```bash
pip install jupyterlab
```

## How to Use

1. Navigate to this directory in your terminal
2. Launch Jupyter: `jupyter notebook` or `jupyter lab`
3. Open the notebook you want to work on
4. Follow the instructions in each notebook

## Contributing

We welcome new notebooks! Please follow the naming convention:
- `XX_topic_name.ipynb` where XX is a two-digit number
- Include clear explanations and comments
- Test all code cells before submitting
