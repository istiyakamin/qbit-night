# Quantum Computing Examples

This folder contains example quantum programs demonstrating various quantum computing concepts.

## Available Examples

### Basic Examples
- [hello_quantum.py](hello_quantum.py) - Your first quantum program
- [bell_state.py](bell_state.py) - Creating quantum entanglement
- [quantum_random.py](quantum_random.py) - True random number generation

## Running the Examples

### Prerequisites
Install the required framework:
```bash
pip install qiskit
```

### Running a Script
```bash
python hello_quantum.py
```

## Learning Path

1. Start with `hello_quantum.py` to understand basic circuit creation
2. Move to `bell_state.py` to learn about entanglement
3. Try `quantum_random.py` to see a practical application
